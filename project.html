Frame Bot

Administration Website

Frame bot is a GPT 3.5 powered Large Language Model using Langchain and OpenAi's embeddings with vector storage to create a custom chatbot for any fighting game!

This Administration website was built to house orders and various transactions within a database of 6000+ Kiosks,  5000+ Stores and 10,000+ Orders.

This Chatbot is designed to answer complex questions about frame data, as well as be a tutorial for new players.
